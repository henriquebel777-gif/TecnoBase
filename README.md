Sistema Tecnobase:
Sistema comercial completo para gerenciamento de estoque, clientes e vendas, desenvolvido como projeto integrador com foco em aplicação real de mercado.

 Sobre o Projeto:
O Tecnobase é uma solução desenvolvida para resolver problemas comuns em pequenos e médios negócios, como:
Falta de controle de estoque
Dificuldade no registro de vendas
Desorganização de dados de clientes

O sistema centraliza todas essas informações, oferecendo mais controle, segurança e eficiência.

Tecnologias Utilizadas:

Frontend: HTML, CSS, JavaScript
Backend: Node.js (Express)
Banco de Dados: MySQL
Integração: API REST
Versionamento: Git e GitHub

Arquitetura do Sistema:
O projeto segue o padrão cliente-servidor, dividido em:

Frontend (Interface do usuário)
Backend (Regras de negócio)
Banco de dados (Armazenamento)
⚙️ Funcionalidades
Cadastro de clientes
Cadastro de produtos
Controle de estoque
Registro de vendas
Atualização automática do estoque
Integração via API
 API (Exemplos de Rotas)
GET /produtos
POST /clientes
POST /pedidos
PUT /produtos/:id
DELETE /clientes/:id
 Como Executar o Projeto
 Pré-requisitos
Node.js instalado
MySQL instalado
Git instalado

 Passos:
 
Clonar o repositório
git clone https://github.com/seu-usuario/tecnobase.git

Entrar na pasta
tecnobase

Instalar dependências:

npm install

Rodar o servidor:

npm start
